# 平成プリクラ機 音声アナウンス 原稿

現在は仮としてAI音声（edge-tts / ja-JP-NanamiNeural、rate+15%、pitch+15Hz）で生成しています。
実際に録音し直す場合は、同じファイル名で書き出して `audio/` フォルダ内のファイルを差し替えてください
（コード側はファイル名を参照しているだけなので、差し替えのみでOKです）。

演者イメージ：当時のゲームセンター店員／プリクラ機の呼び込みのような、ハキハキ元気な女性の声。
早口で明るく、語尾を少し上げるイメージ。

| ファイル名 | 再生タイミング | セリフ |
|---|---|---|
| `01_start.mp3` | タイトル画面で「タッチしてスタート」を押した瞬間 | 「レッツ、プリクラ！」 |
| `02_select_curtain.mp3` | 背景・フレーム選択画面に入った瞬間 | 「好きな背景を選んでね！」 |
| `03_select_frame.mp3` | 選択画面に入って約2.7秒後（まだ選択画面にいる場合のみ） | 「フレームも決めたら、OKを押してね！」 |
| `04_shot1.mp3` | 1枚目の撮影カウントダウン開始時 | 「1枚目、いくよー！3、2、1、はい！」 |
| `05_shot2.mp3` | 2枚目の撮影カウントダウン開始時 | 「2枚目！かわいく決めて！3、2、1、はい！」 |
| `06_shot3.mp3` | 3枚目の撮影カウントダウン開始時 | 「3枚目！もっと寄って！3、2、1、はい！」 |
| `07_shot4.mp3` | 4枚目（最後）の撮影カウントダウン開始時 | 「最後の1枚だよ！3、2、1、はい！」 |
| `08_deco_start.mp3` | 落書き画面に入った瞬間 | 「らくがきタイム、スタート！制限時間は3分だよ！」 |
| `09_time_warning.mp3` | 残り時間が10秒を切った瞬間（1回だけ） | 「あと10秒！お急ぎください！」 |
| `10_timeup.mp3` | 制限時間が0になった瞬間 | 「はい、タイムアップ！」 |
| `11_finish.mp3` | 「できあがり！」ボタン→確認で「うん！これでOK♪」を選んだ瞬間 | 「できあがり！」 |
| `12_save.mp3` | 保存画面が表示された瞬間（タイムアップ／できあがり共通） | 「らくがき、かんせーい！保存して、プリントは係の人にお願いしてね！」 |

## 録音時の注意
- 各ファイルはセリフ単体のみ（無音の前後カットでOK、多少の間は詰めすぎない方が機械っぽさが出ます）
- `04〜07`（撮影カウントダウン）は現在4.2〜5.3秒程度の長さで、その尺に合わせて画面上の「3・2・1・ハイ！」の表示速度をコード側で自動調整しています。録音し直して長さが変わった場合は [app.js](../app.js) の `SOUND_DURATION` の値（秒数）も実際の尺に合わせて更新してください。
- ファイル形式はmp3のまま同名で上書きすれば差し替え完了です。
